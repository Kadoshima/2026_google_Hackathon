# BE-032：TasksService.enqueueAnalysisTask

## 担当者
- 萩原

## 指示
- `/internal/tasks/analysis` に POST
- body: `{ analysis_id }`
- OIDC 用 service account を env で指定

## 作業ログ
- `enqueueAnalysisTask` を実装し `/internal/tasks/analysis` に POST するタスクを生成
- body を `{ analysis_id }` の JSON として base64 エンコードして設定
- OIDC 用 `TASK_SERVICE_ACCOUNT_EMAIL` を使った `oidcToken` 設定を追加
