# BE-035：POST /v1/upload（GCS保存→Firestore記録）

## 担当者
- 萩原

## 指示
- `StorageService.putRawFile` を呼ぶ
- `FirestoreRepo.createSession/createSubmission` を呼ぶ
- response: `session_id/submission_id/upload_id`

## 作業ログ
- `POST /upload` 内で `StorageService.putRawFile` を呼び出して raw ファイルを保存
- `FirestoreRepo.createSession` / `createSubmission` を呼び出して記録
- response を `session_id` / `submission_id` / `upload_id` で返すようにした
