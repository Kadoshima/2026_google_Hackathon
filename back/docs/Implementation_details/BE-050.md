# BE-050：Vertex AIクライアント雛形（runPrompt/timeout/retry）

## 担当者
- 萩原

## 指示
- `runPrompt<T>(prompt, schema)` の箱を作る
- timeout/retry の設定だけ先に
- 実際のAPI呼び出しは stub でもOK

## 作業ログ
- `src/services/llm/vertex.client.ts` に `runPrompt<T>(prompt, schema, options)` を実装
- `timeoutMs / maxRetries / retryDelayMs` の設定と指数バックオフ付き retry を実装
- 実API呼び出しは `setVertexTransport` で差し替える stub 構成にした
- 追加でモデル自動フォールバックを実装（例: 主モデル404時に次候補へ切替）
- `VERTEX_FALLBACK_MODELS` 環境変数に対応（未設定時は built-in fallback chain を使用）
- フォールバック発生時に `llm_model_fallback` ログを出力するようにした
